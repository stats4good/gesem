Funcionalidades

Sempre que você quiser começar uma nova linha insira duplo espaço na linha imediatamente anterior.

Esta é uma palavra em itálico
Esta é uma palavra em negrito
Esta é uma palavra em negrito e itálico

Para fazer um lista basta usar o sinal de adição:

  • Item 1
  • Item 2
    • Item 2.1
  • Item 3
    • Item 3.1
      • Item 3.1.1

Para fazer um lista numerada basta usar os próprios números:

  1. Item 1
  2. Item 2
    2.1. Item 2.1
  3. Item 3
    3.1. Item 3.1

Para incluir uma figura use a seguinte:

legenda da figura

Para incluir uma equação no texto (inline equation) use \(x_i = \sqrt{\frac{1}{y^3}}\). Note que podem ser utilizados os mesmos comandos que utilizamos em LaTeX.

Para incluir um bloco de equações, o método é o mesmo do LaTeX:

\[\begin{equation} x_i = \sqrt{\frac{1}{y^3}}. \end{equation}\]
